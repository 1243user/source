 1、安装nodejs环境，下载nodejs,直接双击安装（nodejs ,npm 安装ok）

          2、检测有没安装成功：node -v  ,  npm -v

3、淘宝npm镜像（非必要）
	npm install -g cnpm --registry=https://registry.npm.taobao.org

4、安装 vue cli： npm install -g vue-cli

          5、vue -v查看版本号，返回则成功

6、安装webpack :   npm install -g webpack

7、生成vuejs cli项目：   vue init webpack 项目名称

8、进入项目目录： cd 项目名称

9、安装项目的依赖：  npm install

10、启动项目    ：   npm run dev
        安装ajax的插件：　　npm install --save-dev vue-resource
11、生产打包    ：   npm run build
